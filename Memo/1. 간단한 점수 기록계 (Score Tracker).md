#### **1. 간단한 점수 기록계 (Score Tracker)**

## 배우는 핵심 개념##

* 단일 상태 관리 심화 :  firstOperand 대신 currentScore라는 단일 변수의 상태를 관리합니다.

* 증가/감소 이벤트: 하나의 이벤트 핸들러로 여러 버튼(+1, +5, -1, -5)을 처리하는 방법을 연습합니다.

* 경계 조건 처리: 점수가 0 미만이 되지 않도록 막는 (Underflow 방지) 로직을 연습합니다.

**주요 기능**

현재 점수를 표시하는 레이블(Label)  -> currentScore

점수를 1점씩 올리는 버튼 (+1)

점수를 1점씩 내리는 버튼 (-1)

점수를 0으로 초기화하는 버튼 (Reset)


#### **2. 단위 변환기 (Unit Converter)**

배우는 핵심 개념

* 입력 유효성 검사: 숫자가 아닌 문자가 입력되지 않도록 TextBox의 입력을 확인하고 경고하는 방법을 배웁니다.

* 공식 적용 및 데이터 타입: int 대신 double이나 decimal을 사용하여 소수점 계산을 정확하게 처리하는 방법을 익힙니다.

* UI 동기화: 입력 창이 변경될 때마다 결과가 실시간으로 업데이트되도록 이벤트를 연결합니다.


**주요 기능**

섭씨(Celsius) 입력 TextBox
화씨(Fahrenheit) 출력 TextBox
변환 공식: $\\text{F} = \\text{C} \\times 1.8 + 32$




#### **3. 간단한 할 일 목록 (Simple To-Do List)**

배우는 핵심 개념

리스트(List) 상태 관리: 단순히 하나의 숫자 변수가 아닌, List<string> 같은 컬렉션을 상태로 관리하고 조작하는 방법을 배웁니다.
UI와 데이터 동기화: 사용자가 항목을 추가/삭제할 때마다 ListBox와 내부 List<string> 데이터를 일치시키는 방법을 연습합니다.

주요 기능


- 새로운 할 일을 입력하는 TextBox

- 항목을 추가하는 버튼 (Add)

- 할 일 목록을 표시하는 ListBox

- 선택된 항목을 목록에서 제거하는 버튼 (Remove)


#### **4. 아날로그/디지털 시계 (Simple Clock)**

배우는 핵심 개념

* 시스템 이벤트 활용: 사용자 입력이 아닌, 시스템에서 발생하는 주기적인 이벤트 (Timer 컴포넌트)를 사용하여 코드를 실행하는 방법을 배웁니다.

* 시간 및 문자열 서식 지정: $\\text{DateTime.Now}$ 객체를 다루고, 시간을 원하는 형식(예: $\\text{HH:mm:ss}$)으로 포맷팅하여 표시하는 방법을 익힙니다.

**주요 기능**

- 1초마다 DateTime.Now를 가져와 업데이트하는 Timer 컴포넌트

- 현재 시간을 표시하는 레이블 (TextBox 대신 레이블이 더 적합)

* 시계의 시작/정지를 제어하는 버튼 (Start/Stop)
