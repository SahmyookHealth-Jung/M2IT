**#### \*\*1. 간단한 점수 기록계 (Score Tracker)\*\***



**## 배우는 핵심 개념##**



**\* 단일 상태 관리 심화 :  firstOperand 대신 currentScore라는 단일 변수의 상태를 관리합니다. -> 후** UpdateScoreDisplay() 활용하여 로직 분리



**\* 증가/감소 이벤트: 하나의 이벤트 핸들러로 여러 버튼(+1, +5, -1, -5)을 처리하는 방법을 연습합니다.**



**\* 경계 조건 처리: 점수가 0 미만이 되지 않도록 막는 (Underflow 방지) 로직을 연습합니다.**



**\*\*주요 기능\*\***



**현재 점수를 표시하는 레이블(Label)  ->** UpdateScoreDisplay()



**점수를 1점씩 올리는 버튼 (+1, +5)**



**점수를 1점씩 내리는 버튼 (-1, -5)**



**점수를 0으로 초기화하는 버튼 (Reset)**



* **bool에 대한 설명과 bool isHomeSelected = true;에서 true 부분 설명**

역할 : bool 타입은 참(true)과 거짓(false)이라는 두 가지 값만 가질 수 있는 데이터 타입입니다.

사용 목적 : 프로그램의 상태나 조건을 이분법적으로 판단하고 기록할 때 사용합니다.

EX) 로그인했는가? (isLoggedIn) / 게임이 실행중인가? (isRunning)

= true 부분은 프로그램이 처음 시작될 때, 디폴트 값으로 홈팀 (true)이 선택된 상태로 시작하도록 초기값을 설정하는 것입니다.

EX)

private void btnHome\_Click(object sender, EventArgs e)

{

    isHomeTeamSelected = true;

    UpdateTeamSelectionUI();

}



private void btnAway\_Click(object sender, EventArgs e)

{

    isHomeTeamSelected = false;

    UpdateTeamSelectionUI();

}
