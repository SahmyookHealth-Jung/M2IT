* &nbsp;enum 키워드는 열거형을 표현하며, 임의의 숫자로 표현하는 것보다 가독성이 좋기때문에 

정해진 범위에 있는 내용을 열거형으로 나타냅니다.

* enum \[열거형 이름]과 같은 형식으로 사용할 경우 \[열거형 이름]은 int, float, string과 같은 데이터 형과 같이 동작합니다.

따라서 위에서는 Operators라는 열거형은 선언하고, 열거형 변수로 currentOperator를 사용합니다. 

열거형 변수에 값을 넣을 때는 Operators.None, Operators.Add 등과 같이 열거형 내부에 있는 이름을 호출해 사용합니다.

초기에는 연산자가 선택되지 않은 상태를 표현하기 위해 Operators.None으로 값을 초기화합니다.



* Boolean operatorChangeFlag = false; 문장은 연산자 버튼이 눌리면 기존에 입력한 숫자를 저장하고, 

새로 숫자를 입력받기 위한 변수로 사용합니다. 

firstOperand와 secondOperand는 피연산자를 의미하며, 추후 =버튼을 누르면 두 피연산자가 선택된 연산자에 따라 

연산 결과를 출력하는 목적으로 사용됩니다.
